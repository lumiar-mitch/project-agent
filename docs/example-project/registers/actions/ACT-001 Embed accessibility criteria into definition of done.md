# ACT-001 — Embed accessibility acceptance criteria into the definition of done

## What and why

Add explicit WCAG 2.1 AA acceptance criteria to the definition of done for every form and page
template. This shifts accessibility checking left — into each sprint — instead of relying on a single
audit three weeks before go-live. Directly mitigates [RSK-001].
